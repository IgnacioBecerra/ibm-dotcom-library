## Stable selectors

See
[our README](https://github.com/carbon-design-system/carbon-for-ibm-dotcom/tree/main/packages/web-components#stable-selectors-for-analytics-and-integratione2e-testing-in-web-components)
to see how Web Components selector and `data-autoid` should be used.

| Web Components selector               | Compatibility selector                             | Description |
| ------------------------------------- | -------------------------------------------------- | ----------- |
| `<c4d-content-item-horizontal>`       | `data-autoid="cds--content-item-horizontal"`       | Component   |
| `<c4d-content-item-horizontal-media>` | `data-autoid="cds--content-item-horizontal-media"` | Component   |
| `<c4d-content-item-heading>`          | `data-autoid="cds--content-item-heading"`          | Component   |
| `<c4d-content-item-horizontal-copy>`  | `data-autoid="cds--content-item-horizontal-copy"`  | Component   |
